# 最終設計仕様書：Project G-Runner (Tactical SD Refit Action)

## 1. プロジェクト概要

* **タイトル:** Project G-Runner
* **ジャンル:** ハイパーカジュアル・SFゲートランナー（2D）
* **プラットフォーム:** React Native (iOS / Android)
* **開発環境:** Expo, react-native-game-engine, matter-js
* **ビジュアルコンセプト:** **「SDメカ × リアルボス」**。親しみやすいデフォルメメカが、威圧的な巨大兵器に挑むコントラストを強調。

---

## 2. ユニーク・セールスポイント (USP)

* **リアルタイム換装:** ゲート通過により数値だけでなく「機体シルエット」と「攻撃スタイル」が劇的に変化する。
* **フォーム・チェンジ:** 特定のコンボ達成により、一定時間無敵の「覚醒モード」へ移行。
* **ビジュアルコントラスト:** 道中の敵はかわいらしいSDデザイン、ボスはリアルで巨大な脅威として描写。

---

## 3. ゲームシステム仕様 (拡張版)

### 3.1 ゲート・バリエーション

ゲートは以下の3種類に分類され、ランダムに出現します。

| ゲート種別 | 内容 | 効果の例 |
| --- | --- | --- |
| **強化ゲート** | 純粋なステータスUP | `ATK +10`, `FireRate x 1.2` |
| **換装(変形)ゲート** | 武装と**外見（SDモデル）**の総入れ替え | `LongRange (速射) ↔ Heavy (単発高火力)` |
| **トレードオフ** | リスクとリターン | `ATK +50 / Speed -20%`, `HP +30 / ATK -10` |

### 3.2 フォーム・チェンジ (コンボシステム)

* **仕様:** 強化系ゲートを3回連続で（ダメージを受けずに）通過すると、機体が **「オーバードライブ形態」** に変形。
* **ビジュアル:** SDモデルがより煌びやかで、エフェクトを纏った姿に変化。
* **効果:** 10秒間、攻撃速度が最大化し、敵への衝突ダメージを無敵化する。

### 3.3 アクティブスキル：『EXバースト』

* **発動条件:** 敵を倒すかゲートを通ることで「エネルギーゲージ」が蓄積。100%で画面右下のボタンが点灯。
* **効果:** 画面内の全敵ユニットへの一斉射撃（ファンネル/マルチロックオン演出）。

---

## 4. ビジュアル・UX指針（重要）

### 4.1 キャラクターデザイン

* **プレイヤー・道中の敵:** **SD（スーパーデフォルメ、約2〜3頭身）** スタイル。ガンダムのような二足歩行メカや機械生命体を、かわいらしく、かつメカニカルに描写。
* **ボス:** **リアル（頭身が高い、または巨大な非人間型）** スタイル。道中のSDメカとは一線を画す、圧倒的なディテールと威圧感を持たせる。

### 4.2 SF演出

* **背景:** 現代都市や廃墟、宇宙空間をベースに、ネオンやホログラムが走るサイバーパンク/SFテイスト。
* **エフェクト:** ビーム、爆発、バーニアの火などは、ネオンカラー（ブルー、グリーン、ピンク）のGlowエフェクト（光彩）を多用し、デフォルメされた世界観に華やかさを加える。

## 6. CodingAgent への最終指示プロンプト

このブロックをコピーして、Cursor や Claude などの CodingAgent に入力してください。

> **System Prompt:**
> 「React Native (Expo) を使用し、以下の仕様を持つ 2D SF アクションゲームのプロトタイプを作成してください。
> **ビジュアルコンセプト（最重要）:**
> * **『SDメカ × リアルボス』**の世界観。
> * プレイヤー機体および道中の敵キャラクターは、**SDガンダムのようなデフォルメされた（2〜3頭身の）デザイン**にしてください。
> * ステージ最深部のボスキャラクターのみ、リアルで威圧的なデザインにしてください。
> * 全体的な色使いはネオンカラーを多用したSFテイスト（ネオンブルー、黒、グレー）。
> 
> 
> **コアロジック:**
> 1. `react-native-game-engine` による 60fps の縦スクロールランナー。
> 2. **ゲートシステム:** 強化(単なる加算)、換装(武器タイプと**SDモデル**変更)、トレードオフ(ATK増/SPD減など)の3種を実装。
> 3. **コンボ変形:** 強化ゲートを3回連続で通ると、機体がより強力な『Awakened形態（SDモデル）』に変化するロジック。
> 4. **EXバースト:** 敵撃破で溜まるゲージと、全画面攻撃のアクティブスキルボタンを実装。
> 
> 
> **タスク:**
> まずは `GameScreen` の基本コンポーネントと、スワイプ移動・ゲート判定・**SDモデルの換装**の最小プロトタイプを作成してください。素材はシンプルなプレースホルダーで構いませんが、SDスタイルを意識した形状にしてください。」
